# 参与贡献

感谢你愿意改进 **AE2: Utility**。

## 报告问题

- 请尽量说明：**MC 版本**、**NeoForge 版本**、本模组版本、**相关模组列表**（AE2、JEI、ExtendedAE Plus 等及版本）。
- 附上复现步骤；如有日志，请贴出 **latest.log** 中与错误相关的片段。

## 提交代码

- 请先 **fork** 仓库，在分支上修改后发起 **Pull Request**。
- 保持改动与 PR 描述聚焦于单一主题，便于审阅。
- 提交前请在本地执行：`./gradlew build`（或至少 `./gradlew compileJava`），确保能通过编译。

## 代码风格

- 与现有代码保持一致：命名、缩进、注释语言（中文注释用于玩家可见逻辑说明时可保留）。
- 避免无关大范围格式化或顺带重构。

## 法律与许可

- 你提交的代码将按仓库根目录的 [LICENSE](LICENSE)（MIT）授权。
- 若引入第三方代码，请注明来源与许可证兼容性。
